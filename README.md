# 🛒 E-Commerce Frontend (React)

This project is a fully functional **E-Commerce Frontend Application** built using **React.js**.
It was developed over **15 days** with a strong focus on **component-based architecture, state management, and UI design**.

---

## 🚀 Features

* 🛍️ Product Listing Page
* 🔍 Product Details View
* 🛒 Add to Cart Functionality
* ❌ Remove from Cart
* 🔄 Dynamic UI Updates
* ⚡ Reusable Components
* 📦 Context API for State Management

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS
* Context API

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/raushanwith-code/your-repo-name.git
   ```

2. Navigate to project folder:

   ```bash
   cd E-Commerce
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the app:

   ```bash
   npm start
   ```
